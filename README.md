🚀 PriceWatch — Cloud Price Tracking & Alert System
PriceWatch is a cloud-ready price monitoring platform that lets users track products, view price history, and get alerts when prices drop. This project was built as part of cloud platform programming and showcases cloud integration, backend APIs, and price monitoring logic.

🧠 🔍 Overview
PriceWatch solves a common problem: products change price often, and manually checking them is tedious. With PriceWatch, users can:

Add product URLs to track
Fetch and store price history
View highest, lowest, and recent prices
Set price thresholds for alerts
Receive email notifications when target price is reached
(This approach is similar to other price tracking systems that provide history + alert capabilities.

📦 Features
✅ Fetch prices from eBay & Shein APIs
✅ Compare prices across platforms
✅ Store historical price data
✅ Price drop alerts via email
✅ REST API backend
✅ Cloud deployment with AWS Elastic Beanstalk
✅ Simple UI with product listing

🛠 Tech Stack
Layer	Technology
Backend	Python / Django
Database	SQLite / DynamoDB (configurable)
API	REST API endpoints
Frontend	HTML / CSS / JS
Cloud	AWS Lambda / API Gateway (optional)
Alerts	Email notifications via SMTP
Deployed	Elastic Beanstalk
